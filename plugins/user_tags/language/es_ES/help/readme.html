Documentación

Funcionamiento

Si un usuario tiene los derechos suficientes y si las pintadas son fijadas sobre la página " picture " entonces verá aparecer un icono seleccionable para editar las etiquetas de la imagen.

Permisos

Los permisos se definen sobre los estatutos usuarios (invitado, genérico, visitador, administrador, webmestre). Para que un usuario pueda efectuar la acción (añadido, supresión), debe tener un estatuto por lo menos igual al estatuto definido.

Por ejemplo, si se define que sólo los usuarios que tendrán el estatuto visitante podrán añadir pintadas, entonces los usuarios que tienen los estatuos invitado y genérico no podrán aumentar de etiquetas.

El hecho de restringir los derechos sobre los que puede suprimir etiquetas agît sólo sobre la supresión de las etiquetas ya existentes. Si el usuario trata de añadir nuevas pintadas, podrá suprimirlos antes de actualizar.
